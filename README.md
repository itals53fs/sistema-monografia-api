### Dentro da pasta server

Instalando dependencias
```
pip install djangorestframework
```
Subindo o server
```
python3 manage.py runserver
```

### Dentro da pasta resoucres

Instalando dependencias
```
npm i
```

Subindo o frontend

```
npm run serve
```
